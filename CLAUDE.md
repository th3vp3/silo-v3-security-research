# Silo Protocol V3

A Foundry-based Solidity project for the Silo DeFi lending protocol.

## Structure
- `src/` — Smart contract implementations
- `test/` — Test suites
- `script/` — Deployment scripts
- `lib/` — Dependencies (forge-std, openzeppelin, etc.)
- `silo-contracts-v3/` — V3 contract implementations
- `Silo_V3_Security_Report/` — Documentation and quality reports

## Development
Build and test with Forge. See foundry.toml for configuration.
